Frequently Asked Questions

Q: How do I create a collaborative commitment?

A: When creating a new commitment, select "Collaborative" as the type. After creation, you'll receive a 6-digit share code that you can share with friends.

Q: How do I join a friend's commitment?

A: Tap the menu icon, select "Join Challenge", and enter the 6-digit code your friend shared with you.

Q: Is my data private?

A: Yes! Self commitments are stored only on your device. Collaborative commitments are shared only with people you explicitly invite via share codes. See our Privacy Policy for details.

Q: Can I edit or delete a commitment?

A: Yes! Tap the edit icon (pencil) on any commitment card to edit the title, or tap the delete icon (trash) to remove it completely.

Q: Can I check in for past dates?

A: Yes, you can mark completions for any past date by tapping on the calendar day. However, you cannot check in for future dates.

Q: How do I switch between different users?

A: Tap the user switcher at the top of the commitments list to select or create a different user profile.

Troubleshooting

App is not syncing collaborative commitments

  • Check your internet connection
  • Pull down on the commitments list to manually refresh
  • If the issue persists, try closing and reopening the app

Share code is not working

  • Verify the code is exactly 6 digits
  • Make sure you're entering numbers only, not letters
  • Check with the person who shared it to confirm the code is still valid
  • Share codes expire after 7 days of inactivity

Calendar is not showing my completions

  • Navigate to the correct month using the arrow buttons
  • For collaborative commitments, ensure you're online to see others' completions
  • Try refreshing by pulling down on the commitments list
